Azure Resource Profile Specifications (for Galxi Node Schemas)
Accurate, API-aligned fields to enable future Azure integration
Version 1.0 – Covers Compute, Networking, Storage, Databases, Integration, Monitoring & Security.
GLOBAL BASE FIELDS (APPLY TO ALL RESOURCES)

Every Azure resource shares these ARM-level properties.
They must exist on every profile:

Field	Description
name	Azure resource name ("vm-prod-01")
resourceType	ARM type ("Microsoft.Compute/virtualMachines")
subscriptionId	Azure Subscription GUID
resourceGroup	The resource group it belongs to
location	Region (eastus, westus2, etc.)
tags	Key-value metadata
id	Full ARM ID (optional for now, required later)
provisioningState	Succeeded / Failed / Updating
notes	Free-text notes
